# Disparus Org - User Manual

## A. For the Public (Guests / Members)

### 1. Viewing Disappearances
*   **Home Page:** Displays the latest reports.
*   **Search:** Use the search bar to find a name or city.
*   **Filters:** Click "Filters" to refine by:
    *   Status (Missing, Found, etc.)
    *   Type (Animal, Human)
    *   Date of disappearance.
*   **Map:** Click the "Map" tab to view locations.

### 2. Reporting a Disappearance
*   Click the **"Report"** button.
*   Fill out the form accurately:
    *   **Name/First Name:** Mandatory.
    *   **Date:** Approximate date and time.
    *   **Location:** Use geolocation or enter a precise address.
    *   **Description:** Physical details (height, color, distinctive signs).
    *   **Photo:** Add a clear photo (JPG/PNG).
*   Submit. Your report will be subject to moderation.

### 3. Helping with Searches
*   On a "Missing" profile, you can:
    *   **Download PDF Poster:** Print it and post it in your neighborhood.
    *   **Share on Socials:** Download the image optimized for Facebook/Instagram.
    *   **Contact:** Use the contact button to send info (anonymous or not).

## B. For Administrators & Moderators

### 1. Back-Office Access
*   Log in via `/admin/login`.
*   The dashboard displays key stats.

### 2. Moderation of Reports
*   Go to **"Reports"**.
*   **Pending:** List of new reports to verify.
*   **Actions:**
    *   **Validate:** Publishes the report on the site.
    *   **Reject:** Deletes the report (with reason).
    *   **Edit:** Correct typos or erroneous info.

### 3. User Management
*   Go to **"Users"**.
*   You can view the list of registered users, change their role (promote a member to moderator), or ban a malicious user.

### 4. Site Configuration
*   Go to **"Settings"**.
*   Modify social links, contact email, or welcome messages without touching the code.
