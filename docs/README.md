## À propos de la couverture géographique
La plateforme est conçue pour couvrir l'ensemble du continent africain (54 pays) et plus de 650 villes référencées, incluant :
*   **Afrique de l'Ouest** : Sénégal, Côte d'Ivoire, Mali, Bénin...
*   **Afrique Centrale** : Cameroun, RDC, Gabon...
*   **Afrique du Nord** : Maroc, Algérie, Tunisie...
*   **Et toutes les autres régions.**
