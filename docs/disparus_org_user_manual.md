# Disparus Org - Manuel Utilisateur

## A. Pour le Public (Utilisateurs Non-Connectés / Membres)

### 1. Consulter les Disparitions
*   **Page d'Accueil :** Affiche les derniers signalements.
*   **Recherche :** Utilisez la barre de recherche pour trouver un nom ou une ville.
*   **Filtres :** Cliquez sur "Filtres" pour affiner par :
    *   Statut (Disparu, Retrouvé, etc.)
    *   Type (Animal, Humain)
    *   Date de disparition.
*   **Carte :** Cliquez sur l'onglet "Carte" pour voir les localisations.

### 2. Signaler une Disparition
*   Cliquez sur le bouton **"Signaler"**.
*   Remplissez le formulaire avec précision :
    *   **Nom/Prénom :** Obligatoire.
    *   **Date :** Date et heure approximative.
    *   **Lieu :** Utilisez la géolocalisation ou entrez une adresse précise.
    *   **Description :** Détails physiques (taille, couleur, signes distinctifs).
    *   **Photo :** Ajoutez une photo claire (JPG/PNG).
*   Validez. Votre signalement sera soumis à modération.

### 3. Aider aux Recherches
*   Sur une fiche "Disparu", vous pouvez :
    *   **Télécharger l'Affiche PDF :** Imprimez-la et affichez-la dans votre quartier.
    *   **Partager sur les Réseaux :** Téléchargez l'image optimisée pour Facebook/Instagram.
    *   **Contacter :** Utilisez le bouton de contact pour envoyer des infos (anonyme ou non).

---

## B. Pour les Administrateurs & Modérateurs

### 1. Accès au Back-Office
*   Connectez-vous via `/admin/login`.
*   Le tableau de bord affiche les stats clés.

### 2. Modération des Signalements
*   Allez dans **"Signalements"**.
*   **En Attente :** Liste des nouveaux signalements à vérifier.
*   **Actions :**
    *   **Valider :** Publie le signalement sur le site.
    *   **Rejeter :** Supprime le signalement (avec motif).
    *   **Modifier :** Corrigez les fautes ou les infos erronées.

### 3. Gestion des Utilisateurs
*   Allez dans **"Utilisateurs"**.
*   Vous pouvez voir la liste des inscrits, changer leur rôle (passer un membre en modérateur), ou bannir un utilisateur malveillant.

### 4. Configuration du Site
*   Allez dans **"Paramètres"**.
*   Modifiez les liens sociaux, le mail de contact, ou les messages d'accueil sans toucher au code.
