# Manuel Utilisateur

Ce guide explique comment utiliser la plateforme DISPARUS.ORG pour rechercher, signaler ou aider à retrouver une personne ou un animal disparu.

## 1. Rechercher une Disparition

La page d'accueil et la page "Recherche" permettent de consulter la base de données.

### Options de Filtrage
*   **Barre de recherche** : Saisissez un nom, un prénom ou une ville.
*   **Type** : Filtrez pour n'afficher que les "Personnes" ou les "Animaux".
*   **Statut** :
    *   *Disparu* : Recherche active.
    *   *Retrouvé* : La personne/animal a été localisé.
    *   *Décédé* : Malheureusement retrouvé sans vie.
*   **Pays** : Filtrez par pays de disparition.
*   **Avec photo** : Cochez pour ne voir que les fiches disposant d'un visuel.

### Carte Interactive
La carte affiche la dernière position connue. Cliquez sur un marqueur pour voir un aperçu et accéder à la fiche détaillée.

## 2. Signaler une Disparition

Si un proche ou un animal a disparu, utilisez le bouton **"Signaler"**.

### Formulaire de Signalement
Remplissez le formulaire le plus précisément possible :
1.  **Informations Personnelles** : Nom, âge, sexe, description physique.
2.  **Photo** : Ajoutez une photo claire et récente (Formats acceptés : JPG, PNG). C'est l'élément le plus important pour la reconnaissance.
3.  **Lieu et Date** : Indiquez la date précise et la localisation (Ville, Pays). Vous pouvez utiliser la géolocalisation automatique si vous êtes sur place.
4.  **Contacts** : Fournissez jusqu'à 3 contacts (téléphone/email) pour être joint.
5.  **Circonstances** : Décrivez les conditions de la disparition (vêtements portés, véhicule, contexte médical...).

*Note : Votre signalement est publié immédiatement mais peut être modéré a posteriori s'il ne respecte pas les règles.*

## 3. Contribuer (Aider aux recherches)

Sur chaque fiche de disparu, vous pouvez apporter votre aide.

### Signaler une Observation
Si vous pensez avoir vu la personne :
1.  Allez sur la fiche concernée.
2.  Cliquez sur le bouton pour ajouter une information.
3.  Indiquez le lieu, la date et les détails de votre observation.
4.  Si possible, ajoutez une photo "Preuve" (discrète et respectueuse).

### Partager la Fiche
La diffusion est clé. La plateforme génère automatiquement des outils de partage :
*   **Télécharger l'affiche (PDF)** : Un document A4 prêt à imprimer pour affichage physique.
*   **Image Sociale** : Un visuel optimisé pour Facebook/Instagram/WhatsApp.
*   **QR Code** : Un code scannable menant directement à la fiche numérique.

## 4. Signaler un Contenu

Si vous constatez une erreur, un contenu abusif ou une fiche qui n'a plus lieu d'être :
1.  Cliquez sur le lien de signalement (drapeau ou "Signaler le contenu") en bas de fiche.
2.  Sélectionnez la raison (Information fausse, Contenu inapproprié, etc.).
3.  Les administrateurs recevront une alerte pour vérification.
