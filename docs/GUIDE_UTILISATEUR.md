# Guide utilisateur

Ce guide explique comment utiliser la plateforme DISPARUS.ORG pour signaler une disparition ou contribuer a des recherches.

## Acceder a la plateforme

Rendez-vous sur **disparus.org** depuis n'importe quel navigateur web (ordinateur, telephone, tablette).

La plateforme est disponible en francais et en anglais. Pour changer de langue, utilisez le selecteur de langue en haut de page.

## Page d'accueil

La page d'accueil presente :

1. **Statistiques en temps reel** : nombre de signalements, personnes retrouvees, pays couverts, contributions
2. **Barre de recherche** : pour trouver rapidement une personne
3. **Signalements recents** : les dernieres personnes signalees disparues
4. **Carte interactive** : localisation des disparitions sur le continent africain

## Signaler une disparition

### Etape 1 : Acceder au formulaire

Cliquez sur le bouton **SIGNALER** dans le menu de navigation ou sur le bouton rouge "Signaler maintenant" sur la page d'accueil.

### Etape 2 : Remplir le formulaire

Le formulaire comprend quatre sections :

#### Section Identite

- **Type de personne** : Enfant (0-17 ans), Adulte (18-59 ans), Personne agee (60+)
- **Prenom et nom** : Nom complet de la personne
- **Age** : Age au moment de la disparition
- **Sexe** : Homme ou Femme
- **Photo** : Uploadez une photo recente et claire (formats acceptes : JPG, PNG, GIF, WebP)

#### Section Localisation

- **Pays** : Selectionnez le pays de disparition parmi les 54 pays africains
- **Ville** : Selectionnez la ville (la liste s'adapte au pays choisi)
- **Date et heure** : Date et heure approximatives de la disparition
- **Position GPS** : Cliquez sur la carte pour indiquer le lieu exact, ou utilisez le bouton de geolocalisation

#### Section Description

- **Description physique** : Taille, poids, couleur des cheveux, signes distinctifs (cicatrices, tatouages, etc.)
- **Vetements portes** : Decrivez les vetements que la personne portait
- **Objets** : Sac, telephone, bijoux ou autres objets distinctifs
- **Circonstances** : Decrivez le contexte de la disparition (que faisait la personne, ou allait-elle...)

#### Section Contact

Ajoutez jusqu'a 3 contacts qui peuvent etre joints pour toute information :

- **Nom du contact**
- **Telephone** (numero avec indicatif pays)
- **Email** (optionnel)
- **Lien avec la personne** (pere, mere, ami, etc.)

### Etape 3 : Soumettre

Cliquez sur **Soumettre le signalement**. Vous serez redirige vers la fiche de la personne disparue avec son identifiant unique (ex: ABC123).

## Rechercher une personne

### Recherche rapide

Utilisez la barre de recherche en haut de la page d'accueil ou sur la page Recherche. Vous pouvez chercher par :

- Nom ou prenom
- Identifiant unique (ex: ABC123)
- Ville

### Recherche avancee

Sur la page **Rechercher**, utilisez les filtres :

- **Statut** : Tous, Disparu, Retrouve
- **Type** : Tous, Enfant, Adulte, Personne agee
- **Pays** : Filtrer par pays
- **Avec photo** : Afficher uniquement les fiches avec photo

## Consulter une fiche

Cliquez sur une fiche pour voir tous les details :

- Photo et informations d'identite
- Description physique complete
- Circonstances de la disparition
- Carte avec le lieu de disparition
- Contacts pour toute information
- Historique des contributions

### Telecharger des documents

Sur chaque fiche, vous pouvez telecharger :

- **PDF** : Fiche complete imprimable avec QR code
- **Image partage** : Image optimisee pour les reseaux sociaux (1200x630)
- **QR Code** : Pour coller sur des affiches

### Partager

Utilisez les boutons de partage pour diffuser la fiche sur :
- Facebook
- Twitter/X
- WhatsApp
- Par email

## Ajouter une contribution

Si vous avez des informations sur une personne disparue :

### Etape 1 : Trouver la fiche

Recherchez la personne et ouvrez sa fiche.

### Etape 2 : Remplir le formulaire de contribution

Faites defiler jusqu'a la section "Ajouter une contribution" et remplissez :

- **Type de contribution** :
  - *Observation* : Vous pensez avoir vu la personne
  - *Information* : Vous avez une information utile
  - *Retrouve* : La personne a ete retrouvee
  - *Autre* : Autre type d'information

- **Details** : Decrivez precisement ce que vous savez
- **Lieu d'observation** : Si vous avez vu la personne, indiquez ou
- **Date d'observation** : Quand l'avez-vous vue
- **Etat de la personne** : Comment semblait-elle (bien, blessee, desorientee...)
- **Vos coordonnees** : Pour que l'on puisse vous recontacter si besoin

### Etape 3 : Soumettre

Votre contribution sera ajoutee a la fiche. Selon les parametres du site, elle peut etre visible immediatement ou apres validation par un moderateur.

## Signaler un contenu

Si vous constatez une fiche avec des informations fausses ou inappropriees :

1. Ouvrez la fiche concernee
2. Cliquez sur **Signaler ce contenu**
3. Selectionnez la raison :
   - Information fausse
   - Contenu inapproprie
   - Doublon
   - Spam
4. Ajoutez des details si necessaire
5. Laissez votre contact (optionnel)
6. Cliquez sur **Envoyer**

Un moderateur examinera le signalement.

## Conseils importants

### N'attendez pas 24 heures

Contrairement aux idees recues, vous n'avez pas besoin d'attendre 24h pour signaler une disparition. Les premieres heures sont cruciales.

### Preparez-vous

- Gardez des photos recentes de vos proches
- Assurez-vous que vos enfants connaissent au moins un numero de telephone et leur adresse
- Notez les vetements portes par vos proches avant une sortie

### Chaque information compte

Meme si vous n'etes pas certain de votre information, partagez-la. Les enqueteurs et les familles peuvent recouper les temoignages.

## Mode hors-ligne

La plateforme fonctionne partiellement hors-ligne grace a la technologie PWA :

- Les pages consultees recemment restent accessibles
- Vous pouvez installer l'application sur votre ecran d'accueil (telephone)
- Les donnees se synchronisent automatiquement lors de la reconnexion
