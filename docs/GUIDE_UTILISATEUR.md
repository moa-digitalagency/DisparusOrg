# Guide utilisateur

Ce guide explique comment utiliser la plateforme DISPARUS.ORG pour signaler une disparition, rechercher une personne ou contribuer a des recherches.

## Acceder a la plateforme

Rendez-vous sur **disparus.org** depuis n'importe quel navigateur (ordinateur, telephone, tablette).

La plateforme est disponible en francais et en anglais. Pour changer de langue, utilisez le selecteur en haut de page.

## Page d'accueil

La page d'accueil presente :

1. **Statistiques en temps reel** : nombre de signalements, personnes retrouvees, pays couverts, contributions
2. **Barre de recherche** : pour trouver rapidement une personne
3. **Filtre par pays** : menu deroulant pour filtrer par pays (le filtre est sauvegarde)
4. **Signalements recents** : les dernieres personnes signalees disparues
5. **Carte interactive** : localisation des disparitions sur le continent africain

## Signaler une disparition

### Etape 1 : Acceder au formulaire

Cliquez sur le bouton **SIGNALER** dans le menu ou sur le bouton rouge "Signaler maintenant" sur la page d'accueil.

### Etape 2 : Remplir le formulaire

Le formulaire comprend quatre sections :

#### Section Identite

- **Type de personne** : Enfant (0-17 ans), Adulte (18-59 ans), Personne agee (60+)
- **Prenom et nom** : Nom complet de la personne
- **Age** : Age au moment de la disparition
- **Sexe** : Homme ou Femme
- **Photo** : Uploadez une photo recente et claire (formats : JPG, PNG, GIF, WebP)

#### Section Localisation

- **Pays** : Selectionnez le pays parmi les 54 pays africains
- **Ville** : Selectionnez la ville (la liste s'adapte au pays)
- **Date et heure** : Date et heure approximatives de la disparition
- **Position GPS** : Cliquez sur la carte pour indiquer le lieu exact, ou utilisez le bouton de geolocalisation

#### Section Description

- **Description physique** : Taille, poids, couleur des cheveux, signes distinctifs
- **Vetements portes** : Decrivez les vetements que la personne portait
- **Objets** : Sac, telephone, bijoux ou autres objets distinctifs
- **Circonstances** : Decrivez le contexte de la disparition

#### Section Contact

Ajoutez jusqu'a 3 contacts qui peuvent etre joints pour toute information :

- **Nom du contact**
- **Telephone** (numero avec indicatif pays)
- **Email** (optionnel)
- **Lien avec la personne** (pere, mere, ami, etc.)

### Etape 3 : Soumettre

Cliquez sur **Soumettre le signalement**. Vous serez redirige vers la fiche de la personne avec son identifiant unique (ex: ABC123).

## Rechercher une personne

### Recherche rapide

Utilisez la barre de recherche en haut de la page. Vous pouvez chercher par :

- Nom ou prenom
- Identifiant unique (ex: ABC123)
- Ville

### Recherche avancee

Sur la page **Rechercher**, utilisez les filtres :

- **Statut** : Tous, Porte(e) Disparu(e), Retrouve(e), Decede(e)
- **Type** : Tous, Enfant, Adulte, Personne agee
- **Pays** : Filtrer par pays
- **Avec photo** : Afficher uniquement les fiches avec photo

---

## Consulter une fiche

Cliquez sur une fiche pour voir tous les details :

### Informations affichees

- Photo et informations d'identite
- **Compteur de temps** : Affiche depuis combien de temps la personne est portee disparue (jours, heures, minutes)
- Description physique et vetements
- Circonstances de la disparition
- Carte avec le lieu de disparition (zoom niveau 20)
- Contacts pour toute information
- Historique des contributions

### Telecharger des documents

Dans la section "Telecharger / Partager", vous pouvez telecharger :

- **PDF** : Fiche complete avec description, vetements, circonstances et QR code
- **Image** : Image optimisee pour les reseaux sociaux (format portrait 1080x1350px)
- **QR Code** : Pour coller sur des affiches

### Partager

Cliquez sur le bouton **Partager** pour ouvrir le menu de partage avec :

- **Copier le lien** : Copie le lien dans le presse-papier
- **SMS** : Envoyer par message texte
- **WhatsApp** : Partager sur WhatsApp
- **WhatsApp Business** : Partager sur WhatsApp Business
- **Facebook** : Partager sur Facebook
- **X (Twitter)** : Partager sur X
- **LinkedIn** : Partager sur LinkedIn
- **Telegram** : Partager sur Telegram
- **Email** : Envoyer par email
- **Messenger** : Partager sur Facebook Messenger
- **Viber** : Partager sur Viber
- **Pinterest** : Epingler sur Pinterest

Chaque option pre-remplit le message avec "AIDEZ-NOUS A RETROUVER CETTE PERSONNE!" et le nom de la personne.

## Ajouter une contribution

Si vous avez des informations sur une personne disparue :

### Etape 1 : Trouver la fiche

Recherchez la personne et ouvrez sa fiche.

### Etape 2 : Remplir le formulaire de contribution

Faites defiler jusqu'a la section "Ajouter une contribution" et remplissez :

- **Type de contribution** :
  - *Vu la personne* : Vous pensez avoir vu la personne
  - *Info importante* : Vous avez une information utile
  - *Signale police* : Vous avez fait un signalement aux autorites
  - *Personne retrouvee* : La personne a ete retrouvee
  - *Autre* : Autre type d'information

- **Details** : Decrivez precisement ce que vous savez
- **Lieu** : Indiquez le lieu sur la carte (cliquez ou utilisez la geolocalisation)
- **Date d'observation** : Quand l'avez-vous vue
- **Etat de la personne** : Comment semblait-elle (bien, blessee, desorientee...)
- **Photo** : Ajoutez une photo si vous en avez une
- **Vos coordonnees** : Pour que l'on puisse vous recontacter

### Etape 3 : Soumettre

Votre contribution sera ajoutee a la fiche. Selon la configuration du site, elle peut etre visible immediatement ou apres validation par un moderateur.

## Signaler un contenu

Si vous constatez une fiche avec des informations fausses ou inappropriees :

1. Ouvrez la fiche concernee
2. Cliquez sur **Signaler ce contenu**
3. Selectionnez la raison :
   - Information fausse
   - Contenu inapproprie
   - Doublon
   - Spam
4. Ajoutez des details si necessaire
5. Laissez votre contact (optionnel)
6. Cliquez sur **Envoyer**

Un moderateur examinera le signalement.

---

## Comprendre les statuts

Les personnes signalees peuvent avoir trois statuts :

| Statut | Description | Couleur |
|--------|-------------|---------|
| Porte(e) Disparu(e) | Personne toujours recherchee | Rouge |
| Retrouve(e) | Personne retrouvee | Vert |
| Decede(e) | Personne decedee | Gris |

## Conseils importants

### N'attendez pas 24 heures

Contrairement aux idees recues, vous n'avez pas besoin d'attendre 24h pour signaler une disparition. Les premieres heures sont cruciales.

### Preparez-vous

- Gardez des photos recentes de vos proches
- Assurez-vous que vos enfants connaissent au moins un numero de telephone et leur adresse
- Notez les vetements portes par vos proches avant une sortie

### Chaque information compte

Meme si vous n'etes pas certain de votre information, partagez-la. Les enqueteurs et les familles peuvent recouper les temoignages.

### Partagez largement

Plus une fiche est partagee, plus les chances de retrouver la personne augmentent. Utilisez tous les moyens de partage disponibles.

## Mode hors-ligne

La plateforme fonctionne partiellement hors-ligne grace a la technologie PWA :

- Les pages consultees recemment restent accessibles
- Vous pouvez installer l'application sur votre ecran d'accueil
- Les donnees se synchronisent automatiquement lors de la reconnexion

### Installer l'application

Sur mobile :
1. Ouvrez disparus.org dans votre navigateur
2. Appuyez sur le bouton de menu (3 points ou partager)
3. Selectionnez "Ajouter a l'ecran d'accueil"

## Questions frequentes

### Comment modifier une fiche que j'ai creee ?

Contactez l'equipe de moderation via les coordonnees du site. Seuls les administrateurs peuvent modifier les fiches.

### Puis-je signaler une disparition dans un autre continent ?

La plateforme est specialisee sur le continent africain (54 pays). Pour d'autres regions, contactez les autorites locales.

### Mes informations personnelles sont-elles protegees ?

Les coordonnees de contact sont visibles uniquement par les administrateurs et moderateurs. Votre adresse IP est enregistree pour des raisons de securite.

### Comment supprimer une fiche si la personne est retrouvee ?

Ajoutez une contribution de type "Personne retrouvee" ou contactez les administrateurs. Le statut sera mis a jour.
