# Palette's Journal

## 2026-01-27 - The Invisible Foundation
**Learning:** Even well-structured projects often miss the most basic "invisible" accessibility feature: the "Skip to content" link. This forces keyboard users to tab through every navigation item on every page load.
**Action:** Always check `base.html` or equivalent layout files for a skip link first. It's the highest ROI accessibility fix because it's one change that improves every single page.
